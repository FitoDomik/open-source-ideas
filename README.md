## 📋 О проекте

**Платформа идей** — это современный минималистичный веб-сайт, где пользователи публикуют свои идеи для приложений и проектов, а разработчики находят их и воплощают в жизнь.

| 📝 Публикуют идеи | 💻 Реализуют проекты | 🤝 Общаются и растут |
| 🖼️ Добавляют визуал | 🔗 Делятся на GitHub | ⭐ Собирают лайки |

## 🚀 Установка

### 📋 Требования

- PHP 7.4+ (рекомендуется 8.0+)
- MySQL 8.0+ / MariaDB 10.3+
- Apache/Nginx с mod_rewrite
- 128 MB RAM minimum

### ⚙️ Быстрый старт

<details>
<summary><b>1️⃣ Настройка базы данных</b></summary>

```sql
-- В phpMyAdmin выберите вашу базу данных
-- Выполните файл open-source-ideas.sql

-- Затем выполните:
-- database_features.sql (для новых функций)
```

</details>

<details>
<summary><b>2️⃣ Конфигурация</b></summary>

Отредактируйте `config.php`:

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'ваш_пользователь');
define('DB_PASS', 'ваш_пароль');
define('DB_NAME', 'ваша_база');
define('SITE_URL', 'https://ваш-домен.ru');
```

</details>

<details>
<summary><b>3️⃣ Создание директорий</b></summary>

```bash
mkdir -p upload/avatars
mkdir -p upload/files
mkdir -p cache
chmod 755 upload upload/avatars upload/files cache
```

</details>

<details>
<summary><b>4️⃣ Настройка .htaccess</b></summary>

Файл `.htaccess` уже настроен для:
- ✅ SSL редиректа
- ✅ WWW редиректа
- ✅ Защиты файлов
- ✅ Кэширования
- ✅ Сжатия

</details>

## 🎨 API для разработчиков

### AJAX Endpoints

<details>
<summary><b>Лайк идеи</b></summary>

```javascript
fetch('ajax_like.php', {
    method: 'POST',
    headers: {'Content-Type': 'application/x-www-form-urlencoded'},
    body: 'idea_id=123'
})
.then(r => r.json())
.then(data => {
    console.log(data.liked); // true/false
    console.log(data.count); // количество лайков
});
```

</details>

<details>
<summary><b>Добавить в избранное</b></summary>

```javascript
fetch('ajax_favorite.php', {
    method: 'POST',
    body: 'idea_id=123'
})
.then(r => r.json())
.then(data => console.log(data.favorited)); // true/false
```

</details>

<details>
<summary><b>Подписаться на автора</b></summary>

```javascript
fetch('ajax_subscribe.php', {
    method: 'POST',
    body: 'user_id=456'
})
.then(r => r.json())
.then(data => console.log(data.subscribed)); // true/false
```

</details>
